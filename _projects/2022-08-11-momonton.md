# Momonton Todo

브라우저의 LocalStorage와 Geolocation API를 활용한 Todo 페이지입니다. 사용자의 이름과 할 일을 저장하고, 현재 위치 기반 날씨 정보를 함께 표시합니다.

## Demo

[Momonton Todo 열기](/momonton/)

## 주요 기능

- 현재 시간 표시
- 랜덤 배경 이미지
- 사용자 이름 및 이메일 입력
- 할 일 추가/삭제 및 LocalStorage 저장
- 현재 위치 기반 날씨 표시

#### reference

- [HTML Standard Timers](https://html.spec.whatwg.org/multipage/timers-and-user-prompts.html#timers)
    - 타이머의 정확한 실행 시점을 보장하지 않는다.

- [MDN setTimeout 문서](https://developer.mozilla.org/en-US/docs/Web/API/Window/setTimeout#reasons_for_longer_delays_than_specified)
    - 실제 delay가 지정한 값보다 길어질 수 있으며 비활성 탭이나 바쁜 메인 스레드에서 지연될 수 있다.
