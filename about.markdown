안녕하세요. 이곳은 김선영이 컴퓨터 과학, 프로그래밍, 게임 개발을 공부하며 직접 구현한 내용과 시행착오를 정리하는 개인 기술 블로그입니다.

운영자는 한세대학교 컴퓨터공학과 22학번 졸업생으로, C#, C++, .NET, Unity, Unreal Engine, OpenCV, Web DevOps를 중심으로 학습 기록과 프로젝트 회고를 작성합니다. 글은 강의, 문서, 실습, 개인 프로젝트에서 얻은 내용을 바탕으로 작성하고, 참고 자료가 있는 경우 본문에 출처를 함께 남기려고 합니다.

이 블로그는 다음과 같은 주제를 다룹니다.

- 컴퓨터 구조, 네트워크, 데이터베이스 등 전공 학습 정리
- C#, C++, 객체지향 설계, 디자인 패턴
- Unity와 Unreal Engine을 활용한 게임 개발
- OpenCV와 컴퓨터 비전 학습 기록
- GitHub Pages, Jekyll, CI/CD 등 개발 환경 구축

## 글을 작성하는 기준

단순히 검색한 내용을 다시 옮기는 글보다는, 직접 겪은 문제와 해결 과정을 남기는 글을 우선합니다. 예를 들어 개발 환경을 설정하다가 발생한 오류, 프로젝트 구조를 바꾸며 얻은 판단, 게임 기능을 구현하면서 확인한 코드 흐름처럼 나중에 다시 참고할 수 있는 기록을 중심으로 작성합니다.

정확성이 필요한 글은 공식 문서나 원문 자료를 확인한 뒤 정리하고, 아직 실험 중인 내용은 그 한계를 함께 적습니다. 기존 글에서 틀린 부분을 발견하면 수정하거나 보충 설명을 덧붙입니다.
