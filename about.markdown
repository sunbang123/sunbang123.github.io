안녕하세요. 이곳은 컴퓨터 과학, 프로그래밍, 게임 개발을 공부하며 정리한 내용을 모아두는 개인 기술 블로그입니다.

운영자는 김선영이며, C#, C++, .NET, Unity, Unreal Engine, OpenCV와 관련된 학습 기록과 프로젝트 회고를 작성합니다. 글은 직접 학습하고 구현하며 정리한 내용을 바탕으로 작성하고, 참고 자료가 있는 경우 본문에 출처를 함께 남기려고 합니다.

이 블로그는 다음과 같은 주제를 다룹니다.

- 컴퓨터 구조, 네트워크, 데이터베이스 등 전공 학습 정리
- C#, C++, 객체지향 설계, 디자인 패턴
- Unity와 Unreal Engine을 활용한 게임 개발
- OpenCV와 컴퓨터 비전 학습 기록
- GitHub Pages, Jekyll, CI/CD 등 개발 환경 구축

사이트 운영과 관련된 문의는 [문의 페이지](/contact/) 또는 이메일 <ilikeksy99@gmail.com>으로 연락해 주세요.
